# CineMizaaj 🎬

A mood-based Bollywood movie recommendation web app built with HTML, CSS and JavaScript using TMDB API.

## Features
- Select your mood and get 3 Bollywood movie recommendations
- Real movie posters and details from TMDB API
- Fully responsive design

## Tech Stack
- HTML5
- CSS3
- JavaScript
- TMDB API

## Assets
The `assets` folder contains the CineMizaaj logo used in the app.
